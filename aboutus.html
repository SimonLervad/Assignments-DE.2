About Git

Here you will find a list, listing the most important codes for learning Git

Git CommandsDescription
git helpIs a command to display commands regarding git
Git initinitiating git - starts in the folder your git is pointing at
cdChange directory - changes git to another folder
mkdirMake directory - Is not a git command, creates a directory in your folder.
git checkout .--Reloads back to last commit
git add .It tells Git that you want to include updates to files in the next commit
git commit -m "....."Command is used to save your changes to the local repository
git pushPushes/uploads all the local changes to the remote repository
git pullDownloads/Syncronizing all the changes in a remote repository to the local repository
git push helpIs a command to specify other commands regarding "git push"
git branchShows all branches in your repositry, and which branch you are currently in
Git branch "..."Creates a new branch
git checkout --track origin/other_branchIf your are a few people working on same repository, and one of you creates a new branch. In order for the group to "syncronize" and get the branch aswell, they will need to run this command. it creates a new branch and your local repo and tracks it from the remote repo
git checkout other-branchSwitches to another branch
